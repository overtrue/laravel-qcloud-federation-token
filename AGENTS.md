# AGENTS.md

Package: **overtrue/laravel-qcloud-federation-token**

## Purpose
Tencent Cloud COS FederationToken generator for Laravel.

## Compatibility Targets
- Laravel: **^12|^13**
- PHP: **^8.3** (Laravel 13 requires PHP 8.3+)

## Local Development
```bash
composer install
composer test
composer check-style
```

## Release Notes
